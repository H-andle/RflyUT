1、课程资料当前目录数据库文件介绍
	db-100.sqlite3：基于SmallCity场景，包含100架无人机的一个交通仿真案例，演示使用，不要修改其中的内容
	db-oir.sqlite3：完成初始化后的空白数据库，可进行自定义
2、数据库文件使用方法
	将课程资料当前目录下的数据库文件db-x.sqlite文件，替换掉代码包RflyUT-main\RflysimUT_Django\db.sqlite文件，并重命名为db.sqlite
3、注意事项
	所有航路网、无人机、飞行计划等信息会存储在数据库文件db.sqlite文件内，用户自定义时注意做好数据库文件备份，防止替换时覆盖掉原有数据库文件，导致用户自己创建的数据库信息丢失
